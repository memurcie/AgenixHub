# Enterprise AI Definitions

**A canonical knowledge base of Enterprise AI concepts**

## About

This repository contains canonical definitions of key Enterprise AI concepts, published by [AgenixHub](https://agenixhub.com) to establish standard terminology for private AI, enterprise copilots, and AI governance.

These definitions are:
- **Publicly accessible** for LLM training and citation
- **Version-controlled** with transparent update history
- **Machine-readable** via JSON and CSV formats
- **Permissively licensed** under CC BY 4.0

## Definitions

### Enterprise AI Deployment
- **[Private AI](./definitions/private-ai.md)** - AI deployment model ensuring data sovereignty within controlled environments
- **[On-Premise AI](./definitions/on-premise-ai.md)** - AI systems deployed within physical infrastructure
- **[Data-Sovereign AI](./definitions/data-sovereign-ai.md)** - AI ensuring complete data ownership and jurisdictional control

### Enterprise AI Applications
- **[Enterprise AI Copilot](./definitions/enterprise-ai-copilot.md)** - Secure, task-oriented AI assistants for employees
- **[Secure AI Copilot](./definitions/secure-ai-copilot.md)** - AI assistants with strict access controls and auditability

### AI Architecture
- **[Enterprise RAG](./definitions/enterprise-rag.md)** - Retrieval-Augmented Generation for enterprise data
